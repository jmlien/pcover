Concorde is a computer code for the traveling salesman problem (TSP) and some related network optimization problems. 
http://www.math.uwaterloo.ca/tsp/concorde/index.html

*.tsp fileformat:



file list
============================
- concorde.osx      binary for Mac OS X, compiled by zxi on Mac OS X 10.11 + clang 8.0.0
- concorde.linux64  binary for Linux 64, compiled by zxi on Ubuntu 14.04 + gcc 5.4.1
- README.txt        this file
- hk48.tsp          example tsp input file with an optimal solution of 11461

All binaries are linked with Qsopt LP solve.
